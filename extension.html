ARIA Chrome Extension

ARIA follows your browsing with intelligent, privacy-aware support — quick-look answers, trading news, price comparison, and a quiet reminder system that never speaks private details out loud.

Quick ARIA

Open the extension from any tab for an instant answer, search, or tech triage — without leaving the page.

Reminders

Set a task from the extension. When it's due, ARIA only says "You have a task due soon." — never the contents.

Trading News

Aggregated from Yahoo Finance, CoinGecko, and SEC EDGAR — one tap, one view, no clutter.

Price Compare

Send a product query across Google Shopping, Amazon.ca, Best Buy, and eBay Canada in a single click.

Install in Chrome (unpacked)

  1. Download the aria-chrome-extension.zip above and unzip it.
  2. Open Chrome and go to chrome://extensions.
  3. Toggle Developer mode on (top right).
  4. Click Load unpacked and select the unzipped folder.
  5. Pin the ARIA extension to your toolbar for one-tap access.

A Chrome Web Store listing is in review — the unpacked install works today on every Chromium-based browser (Chrome, Edge, Brave, Arc, Opera).
