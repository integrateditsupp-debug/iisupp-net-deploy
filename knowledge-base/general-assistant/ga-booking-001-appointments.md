# Booking appointments online

## 1. Common booking scenarios
- Medical / dental.
- Restaurant.
- Salon / barber / spa.
- Trade services (plumber, electrician).
- Government appointment (DMV, passport).
- Test / exam.

## 2. ARIA's flow
1. Confirm: what kind of appointment, with whom, when, where, who attends.
2. Find the right booking system — usually the business's website or a platform (OpenTable, Resy, Zocdoc, Setmore).
3. Walk user through dates / times available.
4. Fill the form fields with verified info.
5. Confirm & save the confirmation email + add to calendar.
6. Set a reminder.
7. Track cancellation policy + reschedule rules.

## 3. Personal info safety
- Only enter info on legitimate booking site (verify URL).
- Don't book through email links — go to the source.
- Phone number may be required for SMS reminders — use a real number.
- For medical, the patient portal is the right channel for personal/medical info.

## 4. Calendar integration
- Add appointment to user's preferred calendar.
- Travel time before / buffer after.
- Reminder 24h + 1h.
- Note address + parking.

## 5. Reschedule / cancel
- Most platforms let you reschedule 24h+ in advance for free.
- Note the cancellation policy at time of booking.
- Pay attention to penalty terms (e.g., medical no-show fees).

## 6. Confirmation discipline
- Save confirmation email.
- Note confirmation number / appointment ID.
- Bring required docs (ID, insurance card, prep instructions for medical).
