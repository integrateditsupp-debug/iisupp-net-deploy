# Filling forms online safely

## 1. Before starting
- Confirm site is legitimate (URL, padlock, SSL).
- Confirm what info is being collected and why.
- Understand consequences (will it create an account? bind a contract? submit to a government?).

## 2. Field hygiene
- **Required vs optional:** fill required only. Skip optional unless valuable.
- **Sensitive data (SSN, ID, credit card):** only on https sites with reason.
- **Phone / email:** real if needed for confirmation; secondary email if just signing up.
- **Address:** correct format for region; ZIP + 4 if asked.
- **Date of birth:** required for many; leave blank if not.
- **Password:** use unique strong; let password manager generate.

## 3. Autofill caution
- Browser autofill is convenient but can leak: a malicious form can pull hidden fields you weren't shown.
- Review what's about to submit before clicking submit.

## 4. Save evidence
- Screenshot or save the receipt page.
- Save the confirmation email.
- For government: keep tracking ID + reference numbers.

## 5. Government forms specifically
- Triple-check name spelling, SSN, dates.
- Many forms can't be edited after submission.
- Use the official URL only (`.gov`).
- Keep PDFs / printouts in a labeled folder.

## 6. ARIA helps with
- Telling user which fields are required.
- Spotting if a form is asking for something unusual.
- Suggesting whether to submit or pause.
- Saving the receipt to a known location.
