# KB Index

## L1
- No sound from Windows desktop
- Microphone not working in meeting app
- Wi-Fi connected but no internet
- Printer not printing
- Outlook not sending or receiving
- Password reset or locked account

## L2
- VPN connects but internal resources fail
- DHCP failure / 169.254 address
- DNS resolution failure
- Intune device enrollment failure
- Shared mailbox access issue
- Slow computer diagnosis

## L3
- Active Directory domain login outage
- Entra ID / Conditional Access lockout
- Firewall or routing production outage
- Ransomware or malware incident triage
- Data recovery planning after drive failure

## Legacy
- Mapped network drive not connecting
- Old line-of-business app only works on one PC
- On-prem Exchange/old Outlook compatibility issue

## Modern
- SaaS SSO login issue
- Zero Trust device compliance block
- Cloud backup restore request

## Security
- Phishing email response
- Suspicious sign-in alert
- Lost or stolen company device
